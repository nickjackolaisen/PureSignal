Welcome to PureSignal

Follow these steps to get full protection and optional Pro features.

  1. Pin the extension for quick access from your toolbar.
  2. Create an account at puresignal.io (use the email login on the home page).
  3. Open your dashboard, copy your User ID, and paste it into extension Settings.
  4. Subscribe on the pricing page for Pro blocking updates and accountability tools.
  5. Open settings and set a protection password so settings cannot be changed impulsively.
  6. Enable incognito protection in chrome://extensions if you use private windows.

API Base URL is pre-filled as https://api.puresignal.io. You only need to paste your User ID.
